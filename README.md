# Compilation steps

* Install Node.js in your system. If you have *Homebrew*, you can simply execute 

```bash
brew install node@24
brew install pnpm
```

After this please access the folder where the project is stored and execute

```bash
pnpm install
pnpm run dev
```

With this you should be able to connect into the webserver and play with the code. If you do not want to install or execute anything, please access this link where the files are served.
